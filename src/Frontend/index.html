The Grid

What Is a Smart Grid?

The electrical grid is the nervous system of modern civilization, a vast interconnected network that generates, transmits, and delivers electricity across thousands of kilometres. Every factory, hospital, and home depends on it operating in perfect real-time balance.

A smart grid layers sensors, data communications, and AI on top of this physical infrastructure. Instead of reacting to failures, it anticipates them, enabling proactive dispatch, renewable integration, and demand-side flexibility.

The Challenge

A World-Scale Problem
Hiding in Plain Sight

Electricity cannot be stored at scale. Every watt generated must be consumed instantly, making forecasting not just useful, but existentially critical.

01
DV
Demand Volatility
Consumption patterns shift violently intraday. Heatwaves spike AC load by 30% in hours, while industrial shutdowns drop demand without warning.
+40% swing in peak vs. off-peak load
02
RI
Renewable Intermittency
Solar and wind power are inherently unpredictable. Grid operators must compensate in real time or risk collapse.
~35% of new capacity is variable renewables
03
DE
Dispatch Inefficiency
Over-provisioning costly peaker plants to handle uncertainty burns billions annually in unnecessary fuel costs, emissions, and capacity payments.
$150B+ wasted globally on excess reserves
04
DQ
Data Quality Gaps
Meter faults, communication failures, and sensor drift create corrupted demand time-series, rendering naive forecasts dangerously inaccurate.
Up to 8% of raw grid readings are erroneous
05
IS
Infrastructure Stress
Aging transmission infrastructure operates near thermal limits. Unexpected load surges cause cascading failures that can take entire regions offline.
70% of global grid assets are over 25 years old
06
GC

The System

Six Stages. Zero Gaps.

A fully automated pipeline from raw NRLDC portal data to a live operational dashboard, with no manual steps required.

SC
STAGE 01
Scrape
Selenium-powered automation downloads monthly NRLDC Excel files into versioned raw storage.
EX
STAGE 02
Extract
Parser normalizes heterogeneous Excel layouts into a unified 15-min demand Parquet file.
CL
STAGE 03
Clean
Physics-grounded anomaly detection flags spikes above 5,000 MW and out-of-range readings, then interpolates repairs.
ML
STAGE 04
Train
XGBoost model trained on lag, rolling-stat, and calendar features with season-aware 3-month holdout validation.
API
STAGE 05
Serve
Flask API loads model once at startup and serves 96-step autoregressive forecasts in milliseconds.
UI
STAGE 06
Visualize
Operations dashboard renders forecast chart, KPIs, residual heatmap, and CSV export for grid teams.

Technology

Built on Production-Grade Stack

ML
XGBoost + scikit-learn
Gradient-boosted trees trained on rich lag, rolling-stat, and calendar features. Season-aware holdout validation ensures real temporal generalization.
ETL
Pandas + Parquet Pipeline
DatetimeIndex-first data processing with columnar Parquet storage. Physics-grounded anomaly detection with time interpolation repair.
API
Flask REST API
Lightweight inference service with three endpoints: /health, /forecast (96-step), /residuals. Sub-millisecond latency, CORS-enabled, stateless design.
BOT
Selenium Web Automation
Headless Chrome with WebDriver automates NRLDC SPA navigation, handling pagination, dynamic tables, and partial-download detection.
UI
SVG-Powered Dashboard
Zero-dependency single-file frontend. Renders forecast chart, 7x24 residual heatmap, KPI cards, and model comparison tabs in-browser.
OPS
Artifact-Based Model Lifecycle
joblib model + JSON metadata buffer enables instant API cold-start with no online retraining dependency. Clean separation of training and serving.
